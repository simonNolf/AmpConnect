# AmpConnect
A connected dac. 

## how to install

take all the code

open a terminal

go to ```AmpConnect/site/interfaceweb/frontend```

```
npm install

npm update

npm upgrade

npm run build

```
and in ```AmpConnect/site/interfaceweb/backend```

```
npm install

npm update

npm upgrade

npm run dev
```

then, go [there](localhost:8080)
